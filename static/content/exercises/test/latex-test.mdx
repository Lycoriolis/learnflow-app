# LaTeX Rendering Test

This is a test exercise to verify LaTeX rendering capabilities.

## Simple Math

Here's a simple inline equation: $x = 2$

## Block Equation

$$\int_{0}^{1} x^2 dx = \frac{1}{3}$$

## Question

What is the derivative of $f(x) = x^2$?

**Answer:** $f'(x) = 2x$
